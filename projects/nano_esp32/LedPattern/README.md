# LEDs Pattern

I created a circuit with 4 LEDs and controlled them independently to make an alternating pattern effect.

**Components**:
- Arduino Nano ESP32
- 4 LEDs (red, yellow, green and blue)
- 4 resistors (3 220 Ohms and 1 100 Ohms -> Blue LED)
