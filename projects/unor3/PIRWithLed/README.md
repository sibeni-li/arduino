# PIR with LED

It's my first circuit with just a PIR (Passive Infrared) motion sensor and a LED. The LED light on when the PIR detect a movement.

**Components**:
- Arduino UNO R3
- PIR motion sensor (HC-SR501)
- Red LED
- 220Ohms resistor
