# Meteo Station

This is a simple weather station that displays temperature and humidity on an LCD screen, along with messages indicating the comfort level for each range. It also lights up the corresponding LED based on the current temperature range.

**Components**:
- Arduino UNO R3
- DHT11 sensor
- LCD1602 display ()
- 4 LEDs (red, yellow, green, blue)
- 4 resistors 220Ohms
- 10k potentiometer
